# 💍 Amora - Invitación de Boda Digital

Una experiencia interactiva y elegante diseñada para invitar a seres queridos a celebrar la unión de dos personas. Este proyecto reemplaza la invitación de papel tradicional por un ecosistema digital completo, gestionando desde la confirmación de asistencia hasta la lista de regalos.

## ✨ Características Principales

- **Timeline Interactiva:** Un recorrido visual por la historia de la pareja con animaciones al hacer scroll.
- **Gestión de RSVP:** Formulario inteligente que registra invitados, gestiona acompañantes y asigna mesas automáticamente evitando duplicados.
- **Lista de Regalos Dinámica:** Buscador integrado para encontrar artículos por nombre o número y sistema de reserva en tiempo real.
- **Muro de Mensajes:** Espacio dedicado para que los invitados dejen sus deseos, con funcionalidades de Crear, Leer, Editar y Borrar (CRUD).
- **Diseño Responsive:** Optimizado para dispositivos móviles, tablets y computadoras.

## 🚀 Tecnologías Utilizadas

- **HTML5:** Estructura semántica para accesibilidad.
- **CSS3:** Uso de CSS Grid para galerías, Flexbox para alineación y transiciones suaves para una experiencia de usuario (UX) premium.
- **JavaScript (Vanilla):** Lógica de persistencia de datos local, manejo del DOM y validaciones de formularios.

## 📦 Estructura del Proyecto

1. `index.html`: Contenedor principal con navegación y música de fondo.
2. `portada.html`: Bienvenida con cuenta regresiva.
3. `historia.html`: Línea de tiempo con animaciones.
4. `detalles.html`: Información logística y mapa.
5. `programa.html`: Itinerario del evento.
6. `asistencia.html`: Sistema de confirmación (RSVP).
7. `regalos.html`: Catálogo de regalos interactivo.
8. `vestimenta.html`: Guía de Dress Code con referencias visuales.
9. `gracias.html`: Galería de fotos y muro de mensajes.
